# t3rn-exec

Just a simple executor.

## Running

### 1. Clone the Repository
```bash
git clone https://github.com/roberterichgi/t3rn-exec
cd t3rn-exec
```

### 2. Run the Script
Execute the script using the following command:
```bash
chmod +x v53.sh && ./v53.sh
```
